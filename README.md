# AI-Personal-Ops-Copilot

An autonomous, multi-agent orchestrator built in n8n for personal operational efficiency. This system acts as a "Second Brain," automating task delegation, data retrieval, and complex business workflows.

### 🛠 Core Engineering Features
* **Agentic Orchestration**: Built using GPT-4o for intelligent intent classification and autonomous task routing.
* **Context Management**: Implements long-term memory via **Pinecone Vector Databases**, enabling semantic search and context-aware responses.
* **Intelligent Automation**: Handles complex multi-step workflows like email, calendar, and contact management via asynchronous tool calling.
* **State Management**: Uses Window Buffer Memory to maintain conversation continuity, eliminating "Goldfish Syndrome" in agent interactions.

### ⚙️ Technical Architecture
* **Orchestration**: n8n (Event-Driven Workflow Automation).
* **LLM Engine**: GPT-4o for reasoning and Claude 3.5 Sonnet for content generation.
* **Memory & Storage**: Pinecone (Vector Store) for persistent business knowledge.
* **Integration Layer**: Custom API tool calling for Gmail, Google Calendar, and Airtable.

### 🚀 Why this project?
Standard chatbots are stateless and limited. This project bridges the gap between static LLMs and autonomous agents by creating a **resilient, closed-loop system** where agents can communicate, handle errors via retry logic, and execute high-value business operations without manual intervention.
